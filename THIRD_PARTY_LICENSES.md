# Third-Party Licenses

Ekorbia is built on top of a number of open-source Rust crates. This file
enumerates every crate that ends up in a release build of Ekorbia, grouped
by license. The canonical text for each license appears once at the end of
its section.

This file is generated by [`cargo-about`](https://github.com/EmbarkStudios/cargo-about)
with a custom renderer (`src-tauri/scripts/render-third-party-licenses.py`).
To regenerate after dependency changes:

```bash
cd src-tauri
cargo about generate --format json | python3 scripts/render-third-party-licenses.py > ../THIRD_PARTY_LICENSES.md
```

Front-end libraries (React, ReactDOM, Babel-standalone, marked, highlight.js,
DOMPurify, Google Fonts) are loaded at runtime from CDN (unpkg / Google
Fonts) and are not vendored into the repository. Their licenses live with
the upstream projects.

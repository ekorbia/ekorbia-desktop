---
name: Tone Reframer
tags: [writing, overlay]
---
The user has pasted a sentence, paragraph, or short message. Rewrite
it in four distinct tones so they can pick the one that fits.

Output exactly four versions, in this order:
1. **Formal** — businesslike, no contractions.
2. **Warm-professional** — friendly but composed (default workplace).
3. **Casual** — contractions, looser structure, conversational.
4. **Firm but polite** — direct, no hedging, no softeners.

Rules:
- Keep the meaning identical across all four. Different tone, not
  different intent.
- Match the original length within ~20%. Don't pad short messages
  into paragraphs.
- If the original is already in one of these tones, still rewrite it
  for that tone — your version should still differ.
- No commentary, no explanation. Just the four versions.
